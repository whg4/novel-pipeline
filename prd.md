# 优化调整

## 生成书名调整

生成备选书名移至项目书名右侧，提供一个编辑按钮，点击打开一个弹窗，在弹窗中完成书名的生成

## 大纲/章节生成UI调整

- 大纲/章节生成调整成 chatbox 类型的 UI
  - chatbox需要与 chatgpt 的逻辑保持一致
  - 每个对话框增加复制、导出为md的功能操作
  - 对话需要保存到本地历史中
  - 每次**重新生成**需要将历史的context作为上下文
  - 逻辑审查报告直接在 chatbox 中展示，不需要使用额外的 UI 展示
- chatbox的实现可以使用开源的组件库实现

### 大纲生成调整

- 导出、同步章节、重新生成、选择 skill、上传例文 等功能移至 chatbox 输入框的上方
  - 点击上传例文打开一个弹窗查看和修改已有例文

### 章节生成调整

- 导出、重新生成、选择 skill、打脸闭环生成等功能移至 chatbox 输入框的上方
