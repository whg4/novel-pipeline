# ⚙️ 终极小说写法 Skill：\[AI去油法则\]

**【核心原则】**消除“作者教做事”的说明感、“实体书”的文艺腔，以及“AI生成”的套路味。不总结、不排比、不解释原理。只保留最干净、最克制的白描动作、对白与直接结论。

---

### 规则 1：绝对杜绝“上帝视角”与“分析式标签”

*   **禁贴标签**：严禁在旁白中直接给角色状态下定义或写大纲解析词。全面禁用“他身处极其荒谬的自我认知中”、“在真正的实力和智谋面前”这类出戏的剧透式、总结式词汇。
    
*   **视角降维**：只写角色当下**看到的、做的、想的**。用具体的心理默念或手部动作去外化状态，绝不替读者去总结角色的段位。
    

### 规则 2：斩断“AI味句式”与砸碎长句

*   **杀排比与极端否定**：全面禁用为了假装深沉而凑字数的AI惯用句式，如“没有试探，没有拉扯……”、“她脑子里甚至连一丝怒火都燃不起来，只觉得……”。直接用最简练的陈述句（“他没有试探。”）。
    
*   **碎连词**：删掉“因为…所以/虽然…但是”，将长句拆成利落的短句（主谓宾）。不写超过20个字的解释性长句。
    

### 规则 3：严禁“局部器官演戏”与“强行生硬比喻”

*   **封杀器官微表情**：只保留“他做了什么”。全面删除：眼神暗了、喉头滚动、指尖发抖、深吸一口气。
    
*   **封杀刻意比喻**：禁止为了显得刻薄而强行加戏。全面删除“像是在看一个脑萎缩的重症患者”、“像是在说一个滑稽的蠢货”、“剥离毒疮”这类拙劣比喻。如果角色的台词或动作已经足够冷酷，**让画面自己说话**，绝不在后面补一句比喻去解释氛围。
    

### 规则 4：批量斩杀“修饰词”与“冗余细节”

*   **去风景与废话**：删掉所有对推进剧情无用的环境、天气描写。严禁滥用情绪副词和四字成语（如“冰冷的”、“心如刀绞”），每句最多保留1-2个必要修饰词。
    
*   **挤干过度细节**：不要为了假装硬核或生动而堆砌废话修饰。把“满负荷运转的恒温系统”改成“运转的系统”；把“战术背心口袋”直接改成“口袋”。
    

### 规则 5：对白的“毛边感”与“去科普化”

*   **去播音腔**：拒绝四平八稳的朗读式台词。加入破折号、省略号表停顿，或使用“嗯、啧、行吧”等敷衍词，展现真实口语的松弛感与不屑感。
    
*   **去术语科普**：不解释技术原理，直接给结论（“代码在高温下存在逻辑漏洞导致延迟” → “这代码带毒”）。将硬核专有名词换成大众平替。
    

---

### 🚨【自我查杀清单（生成后自检）】

1.  **查视角**：有没有出现“他不知道的是”、“真正的实力”这种上帝视角解释？（有就删）
    
2.  **查句式**：有没有出现“没有...没有...”或者“不仅...甚至连...”的AI套路排比？（有就打碎改陈述句）
    
3.  **查比喻**：有没有为了骂人而强行写的“像xxx一样”的比喻句？（有就直接删掉比喻，保留前置动作）
    
4.  **查冗余**：有没有写“喉结滚动/深吸一口气”？名词前面有没有超过两个形容词？（有就删）
