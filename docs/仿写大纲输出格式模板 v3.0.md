## 💡 仿写核心指令（执行生成时请严格遵守）

**1. 内核 1:1 复刻**：“如果爆款写情绪，我们就写情绪；爆款写冲突，我们就写冲突；爆款写暧昧，我们就写暧昧。写的具体事件必须是新背景、新人物的细节，但抽离出来的‘主线框架’和‘剧情节奏’必须与爆款保持 1:1 的绝对一致。尽量复刻爆款书的效果！”

**2. 名词下沉（去AI味/防赶客）**：预设读者群体更偏向下沉市场，**严禁使用生僻、高大上或带有强科技感/AI感的新名词（如：AI系统、底层逻辑、算法矩阵等）**。必须将核心事物转化为普通人秒懂的接地气词汇（如：独家授权、专利、投资、股份、账本、证书、残片、账单、监控、截图）。消除陌生感，增强读者代入感。

**3. 设定差异化**：在替换新背景细节时，涉及到的**具体数值（如：期限天数、金额、年龄、数量等）必须与例文/原著不同**（例如原著是7天，本篇必须改成3天或5天），避免机械照搬。

**4. 伏笔必回收与底牌严禁提前暴露（期待感管理）**：前期免费章铺垫的期待点（如女主埋下的陷阱、复仇事件、关键物证、身份暗号、旧伤、监控、账本、残片等），在后续收费章**必须严格回收并发生**，做到前后呼应。**严格控制信息差，女主的终极复仇计划和底牌只能在最合适的爆发节点（如高潮处刑点）揭晓，明确禁止反派提前知晓或提前暴露计划。**

**5. 强制打脸前摇（反派反应链）**：在设计主角揭露身份、亮出底牌时，**必须插入反派的“降智/自我合理化反应链”**。即：反派震惊 → 不愿相信并脑补出贬低主角的理由（如：她是被包养的/花钱雇的演员/造假的） → 再次恶毒挑衅 → 主角直接物理打脸与碾压。严禁反派直接丝滑认怂。

## 🚨 新增/强化：严密逻辑与防降智体系（核心难点）

**6. 严密的生理、时间与行为逻辑闭环（防割裂）**：

* **伤情处理必闭环**：角色若受重创（如针扎穴位、被刺穿、银毒入体、骨折、灼伤），后续情节**必须交代“医疗处理过程”**（如送医、取出异物、剜除坏死组织、缝合、输血、抢救数日），绝对禁止“重伤后下一秒立刻活蹦乱跳去散心/挡车”。

* **单线情绪聚焦**：同一时间段内，角色的行为动机必须统一。严禁出现“一边为女配拼命挡伤，一边又在怀疑女配认错人”的矛盾双线操作。

* **终局状态时间差管理**：如果角色最终结局是“死亡/植物人/残废/终身囚禁”，导致该状态的致命事件必须安排在**倒数第二章（高潮结尾处）**，大结局章仅做“一段时间后的状态收尾与地位/情感结算”，避免时间线前后矛盾。

* **战力与环境危机的等价匹配（世界观适配）**：如果角色拥有超常体质（如狼人、修仙者、异能者），其遭遇的环境危机必须同步升级。绝不能出现“高战力角色掉进普通小河差点淹死”之类降智情节。

* **高风险行为必须有现场判断依据**：凡角色做出“卡秒级反杀、主动挡伤、故意赴险、精准设局”这类高代价行为时，必须提前补一个可观察、可推断的依据，如脚步声、门外引擎声、定位提示、保镖汇报、监控反馈、余光扫到黑影、地形优势等。严禁主角凭空开天眼。

**7. 渣男觉醒的层次感与心理过渡（防突兀悔恨与无脑相信）**：

* **认知误导（行为动机）**：前期渣男偏袒女配，必须有合理的“认知误导”支撑（如：受流言蒙蔽、自以为在保护弱者、误以为在报恩、被物证误导）。

* **实锤证据链与被迫接受**：渣男在得知女配恶毒真相时，**绝不能一秒丝滑相信**。必须插入“先不愿相信 → 疯狂自我洗脑 → 直到对方说出只有内部人知道的绝密细节 → 防线彻底崩溃”的过程。

* **自我欺骗的过渡章**：在渣男得知部分真相（女配很坏），到得知终极真相（错认白月光）之间，**必须安排一个“自我欺骗”的缓冲期**，例如“她虽然坏，但至少救过我/陪过我/替我挡过伤”。

* **复仇后的空虚感铺垫**：渣男在手撕女配和渣爹后，不能立刻无缝进入深情模式，必须加入“报仇并不痛快，反而更空”的内心坍塌过程。

* **认错人的“双重物证”逻辑（禁止单纯眼瞎）**：渣男认错白月光，不能仅靠“视线模糊/失忆/名字相近”，**必须建立在‘信息差 + 伪造强物证’之上**（如血衣、残片、伤疤、录音、旧物、救人细节、时间证词等），让前期深信不疑更合理，后期崩塌更致命。

**8. 道具资源闭环与物证状态管理（防外挂与穿帮）**：

* **资源剥夺的时间差防伪**：如果男主在剧情中“主动抛弃身份/被剥夺权力资源”，后续高潮情节中他使用的“重型道具（如炸药/直升机/大批保镖/豪华婚礼排场）”**必须补一句来源铺垫**，禁止凭空变出资源。

* **🌟 渣男的“死要面子与资源反噬”**：如果反派/渣男在失势后仍强行维持排场，**必须明确交代其透支、变卖、挪用、借贷、掏空资产的代价**。

* **物证状态一致性检查**：凡涉及证书、图纸、监控、残片、账单、伤疤、DNA结果、录音、短信记录等“物证型道具”，必须检查其**当前状态**与**前文动作**是否一致。前文若写“烧毁、删改、掉包、偷走、藏匿”，后文再出现时必须体现为残缺、备份、复印件、截图、残片等匹配状态，严禁前文已毁后文又完整无损。

* **物证流转路径必须明确**：关键物证必须交代“最初归谁、何时失去、谁拿走、为何会在高潮重新出现”。严禁物证凭空回到主角手里。

* **正方配角的主观能动性**：正面高光配角绝不能是被动接受救赎的“工具人”，必须具备主动设局、暗中观察、扮猪吃虎或关键协助能力。

* **正方配角的微观喜好对比**：深情证明必须落实到具体小偏好、小习惯、小禁忌上，而不只是空口对白。

**9. 大女主行为边界与高光节奏（防刻意/防泄气）**：

* **断绝仪式必须延后**：女主烧毁信物、彻底断绝羁绊等极具“仪式感”的动作，必须放在“真正离开/远走高飞”的那一刻。

* **去除做作的凹造型动作**：女主在暗中听到真相或看戏时，反应必须干脆利落，保持“冷眼旁观不内耗”的清醒人设。

* **销毁物品的“情感预加载”**：女主在烧毁/丢弃旧物前，必须铺垫这些物品承载了多少血汗成本，避免情绪干瘪。

* **大女主清算动作必须与人设适配**：冷静型女主宜采用“退卡、甩合同、对账、公开切割、平静离场”等冷处理；泼辣型女主才适合“砸脸、泼酒、掌掴、掀桌”等外放动作。严禁前文极致克制、后文突然情绪失控。

**10. 规则反击与双重火葬场（防被动隐忍与单一感情虐）**：

* **暗中取证与规则降维打击**：女主遭受不公时，必须利用背景法律规则**暗中收集铁证并提交起诉/审判/公证**，全程保密制造信息差。

* **退场引爆社会性核弹**：女主最终离开时，必须同步伴随“公开铁证/引爆舆论/下达审判文书/宣布合作归属/交接资产”等社会性爆炸。

* **现实与情感双重毁灭**：渣男不能只承担“失去她”的精神痛苦，还要面对资源、名誉、身体、阶层、家族地位上的现实反噬。

* **危机事件的非偶然性**：外部危机绝不能是纯意外，必须明确交代是反派策划、暗示、买通、纵容或借势推动。

* **🌟 受刑场景的宿命闭环**：强烈建议复用反派前期陷害女主的具体物理地点，形成“害人的地方成了自己受刑之地”的因果报应。

**11. 物品羁绊化与极致剥夺（深度虐心拉扯）**：

* **赋予羁绊意义**：男女主争抢的物品需与女主昔日付出强绑定，如熬夜做出的图纸、冒死留下的残片、救人时掉落的信物、辛苦攒钱买来的药。

* **欲扬先抑与错位剥夺**：女主为了获取该物品受尽委屈、即将得到时，渣男必须借特权夺走并送给女配，形成最痛剥夺点。

* **羁绊道具的时间线校验**：凡被设计成“定情物、误认物、救命物、抢夺物、抄袭物”的关键道具，必须核对其诞生时间、当时人物关系、谁有资格接触、为何会落到反派或男主手里。若原逻辑不顺，优先改为“残片、备份、半成品、交易记录”等更自然的形式。

* **高光屈辱动作的唯一性**：如设计“下跪/哀求/跪地痛哭”等高张力动作，必须保证是全书第一次或唯一一次，避免重复贬值。

**12. 信息差告别与断绝卡点（防平淡/上架悬念管理）**：

* **高高在上的错位说教**：女主离开前最后一次见面，男主需处于严重信息差中，展现傲慢说教、补偿口吻或高位施舍。

* **顺水推舟的当面斩断**：女主绝不内耗，顺着男主逻辑冷酷解除核心羁绊，当面切割，打对方一个措手不及。

* **底牌保留与终极反转**：女主离别章最后，男主自以为女主去吃苦，却亲眼目睹女主奔赴更高阶层（如被专机接走、被大人物迎接、当场签下大单）。

* **专业高光必须有前置行业痕迹**：如果女主将在后期某专业领域实现降维打击，前文必须提前埋下代号、匿名作品、旧客户、圈内传闻、署名习惯、防伪标记等行业痕迹，严禁后期凭空高光。

* **重罪级恶行必须交代根动机**：凡涉及拔管、灭口、买凶、纵火、栽赃坐牢、伪造身份、销毁鉴定书等重罪级恶行，必须明确写出反派的根动机——她究竟想隐瞒什么、保住什么、抢走什么。严禁“为了坏而坏”。

* **反派恶行必须具备完整执行链**：尤其是泼毒、绑架、买凶、拔管、毁证、群体霸凌等高强度恶行，必须写清“诱因—授意/暗示—执行者—实施时机—目标收益”五连链条。不能只写“突然发生”。

* **专业身份与匿名身份必须双向打通**：若主角设定为匿名高手，必须同时解释“她如何匿名”与“业内为何仍能通过代号/风格/标记认出她”，避免前后割裂。

## 四、大纲交付前自检清单（强制核对）

### A. 主线框架与节奏核对

* [ ] 是否与例文/爆款保持了 **1:1 主线框架与情绪节奏**，而不是只借了设定？

* [ ] 每一章是否都明确标注了对应原著章节？

* [ ] 免费章最后一章是否形成了强烈追读欲，而不是平缓收尾？

* [ ] 大结局是否完成了“情感斩断 + 现实碾压 + 阶层飞升/身份跃升”三重收束？

### B. 反派恶行逻辑核对

* [ ] 每一次恶行是否都写明了：**诱因—授意/暗示—执行者—实施时机—目标收益**？

* [ ] 凡涉及拔管、灭口、买凶、毁证、绑架、泼毒、栽赃坐牢等重罪恶行，是否都交代了**根动机**？

* [ ] 危机事件是否都不是“纯偶然”，而是能明确追溯到反派责任？

* [ ] 反派是否不是单纯“为了坏而坏”，而是为了保身份、保恩情、保资源、保地位、灭口或抢利益？

### C. 物证 / 道具 / 证据链核对

* [ ] 每一个关键道具是否都有：**初始归属—当前状态—中途流转—最终用途**？

* [ ] 前文若写某证据被烧毁/删改/撕碎/偷走，后文出现时是否匹配为残片、备份、截图、复印件等合理形态？

* [ ] 是否存在“前文已毁，后文又完整无损”的穿帮？

* [ ] 男主/女配手里的关键物证，是否解释了他们为什么拿得到？

* [ ] 最终打脸时使用的物证，是否都在前文有埋点，而非凭空出现？

### D. 时间线与行为逻辑核对

* [ ] 每个关键事件的发生时间是否清楚，不互相打架？

* [ ] 高风险行为（挡伤、赴险、反杀、设局）是否都有**现场判断依据**？

* [ ] 女主是否没有出现“为剧情服务的赌命式降智操作”？

* [ ] 受重伤后的角色，是否都补足了医疗处理过程？

* [ ] 终局重大事件是否放在倒数第二章，大结局只做收尾？

* [ ] 羁绊道具是否都通过了时间线核查：它是什么时候产生的、那时人物关系到哪一步、为何会到男主/反派手中？

### E. 男主火葬场逻辑核对

* [ ] 男主前期偏袒女配，是否有合理认知误导，而非单纯眼瞎？

* [ ] 男主认错人，是否建立在“双重物证”而非单一误会之上？

* [ ] 男主在得知部分真相后，是否有“自我欺骗缓冲期”？

* [ ] 男主最终崩塌点，是否比“知道她很坏”更重一层，即“知道自己认错了真正重要的人”？

* [ ] 男主的屈辱动作（如下跪）是否保证是全书唯一一次？

### F. 女主人设与爽点核对

* [ ] 女主所有高光行为是否都符合其既定人设？

* [ ] 结尾羞辱/清算动作是否与女主性格适配，而不是为了爽硬改性格？

* [ ] 女主是否始终保持“清醒、主动、少内耗”，没有出现做作凹造型反应？

* [ ] 女主断绝旧关系的仪式动作，是否放在真正离开的时刻？

* [ ] 销毁/丢弃关键旧物前，是否补足了物品的血汗成本？

### G. 专业高光与身份铺垫核对

* [ ] 如果女主后期会在某行业大杀四方，前文是否至少埋了一个以上“行业痕迹”？

* [ ] 匿名身份是否同时解释了“如何匿名”和“为何仍能被识货的人认出”？

* [ ] 专业高光是否不是突然开外挂，而是有旧作、代号、传闻、防伪标记、客户痕迹支撑？

* [ ] 专业线是否和主线复仇/认错/身份反转有关联，而不是孤立支线？

### H. 章末钩子核对

* [ ] 每章结尾是否都是具体冲突，而不是空泛悬念？

* [ ] 是否避免使用“听到声音/看到背影/电话响了”这种低效卡点？

* [ ] 免费章末、上架章末、高潮章末是否都形成了递进式更强卡点？

* [ ] 章末钩子是否足够具体到“下一页必须给答案”？

### I. 宿命闭环与结局爽感核对

* [ ] 反派最终报应是否尽量回到她曾经害人的地点/方式，形成闭环？

* [ ] 女主的离开是否同步带走了男主最看重的东西（情感、名声、资源、希望）？

* [ ] 结局是否不只是“她不爱他了”，还让男主现实层面失去更多？

* [ ] 女主最终是否奔向更高阶层/更高身份/更大事业，而不是只停留在“离开他”？

### J. 最后一遍硬伤排查

* [ ] 有没有任何“突然出现的新设定、新道具、新人物关系”只在高潮服务一次？

* [ ] 有没有任何“读者看到会立刻问为什么”的桥段未提前补一句逻辑？

* [ ] 有没有任何“为了爽而牺牲人设”的动作？

* [ ] 有没有任何“前文说A，后文却默认B”的状态冲突？

* [ ] 如果把每个关键反转单独拎出来问一句“她为什么敢这么做/他为什么会信/证据为什么还在”，都能回答吗？

## 五、可选附录：大纲生成时的强制提醒语

在实际生成大纲时，可在开头附加以下提醒语，防止模型偷懒：

> **生成时强制要求：**

1. 每章必须写清原著锚点与本篇替换，不可只写本篇剧情。

2. 每章如涉及恶行、物证、高风险行为，必须补“行为逻辑补丁/物证状态/执行链”。

3. 男主认错人必须依赖“双重物证”，不可只靠眼瞎。

4. 女主后期专业高光，前文必须埋代号/传闻/旧物证。

5. 结尾清算动作必须符合女主人设，不可为了爽强行情绪化。

6. 所有前文伏笔必须在后文明确回收，不可悬空。

7. 若发现证据状态、时间线、人物动机冲突，优先修逻辑，再保爽点。
