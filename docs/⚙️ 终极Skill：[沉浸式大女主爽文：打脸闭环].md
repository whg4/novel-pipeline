# ⚙️ 终极Skill：\[沉浸式大女主爽文：打脸闭环\]

**【核心目标】**打造极致快节奏、强情绪压迫感、无阅读门槛的“沉浸式爽文”。通过**极简大白话的动作流**替代冗长的文学渲染与器官演戏；通过**具象化的对比与审判者视角**替代空洞的情绪标签；最终通过**“拉爆仇恨→降维打击→战损处刑”**的三步走策略，完成暴爽的打脸闭环。

## 模块二：情绪外化与反差压迫（去空洞）

**原则：不直接写“绝望/恶心/愤怒”，让读者通过强烈的反差画面自己感受。**

*   **规则 5：排比回忆对比（量化牺牲）。**
    
    *   句式公式：他和小三在 \[具体偷情做某事\] 的时候，我在 \[极端环境下做艰难的工作\]。
        
    *   必须用具体的数字（厚度、时间、页数）让心血实体化。_(注：当前3章已用过此手法时，后续需克制，避免重复)_
        
*   **规则 6：心理联想触发“生理性排斥”。**
    
    *   渣男触碰女主时，不要只写“恶心”。必须写出女主**具体联想到了什么肮脏的画面**（如垃圾桶的戒指、用过的避孕套），从而直接引发胃部翻滚、干呕等生理反应。
        
*   **规则 7：假秀恩爱时的“审判者视角”。**
    
    *   当渣男在公众面前立深情人设时，女主不能内耗。必须开启上帝视角：**嘲讽他的虚伪 → 预判他即将一无所有的惨状 → 在心里开启毁灭倒计时。**

## 模块三：冲突推拉与打脸闭环（去憋屈）

**原则：能动手绝不废话，把“憋屈”和“暴爽”的转换时间压缩到极致。**

*   **规则 8：反派挑衅必须“侵犯核心隐私”。**
    
    *   绿茶/小三骑脸输出时，话必须脏、必须绿茶。要直接攻击女主的“床笫私密评价”或“掠夺核心心血的傲慢”，把读者的仇恨值瞬间拉爆。
        
*   **规则 9：物理降维打击，严禁自曝底牌。**
    
    *   面对挑衅，女主绝不能靠大吼大叫或揭露陷阱（“你偷的是废品”）去口舌争胜。
        
    *   **标准连招**：直接物理压制（巴掌/抓头发/砸杯子） → 用上位者身份羞辱（“野狗别冲主人狂吠”） → 冷漠清理（擦手/扔纸巾） → 潇洒离开。
        
*   **规则 10：战损具象化（视觉处刑）。**
    
    *   打完人或曝光真相后，**绝不能一笔带过**。必须给反派一个极度惨烈的“战败特写”（如：精致高定被撕裂、妆容糊成黑泥、肿起的血印、像死狗一样被拖走），完成极致的爽点回收。

## 模块四：章节节奏控制（卡悬念）

*   **规则 11：倒计时压迫与悬念卡点。**
    
    *   在章节开头或转场时，用明确的时间线（“距离晚宴还有48小时”）替代干瘪的“第二天”。
        
    *   每一章的结尾必须短促有力，严禁说破。必须卡在**暧昧拉扯、复仇前摇、倒计时节点**上。

### 🚨【终极强制自查清单（Self-Check）】

每次生成正文后，请用以下5个问题扫描全文：

1.  **查掉价**：女主是不是在跟反派解释原理？是不是在自证清白？（有就改成动作压制）
    
2.  **查仇恨**：反派的挑衅够不够不要脸？有没有踩中隐私痛点？（不够就加料）
    
3.  **查爽点**：反派挨打或身败名裂后，有没有写出他们衣衫不整、涕泗横流的具体惨状？（没有就补特写）
    
4.  **查句式**：有没有超过20个字的解释性长句？（有就敲碎成短平快的动作句）
